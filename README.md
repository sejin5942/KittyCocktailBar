# 🐱🍸 고양이 판타지 쉐이크 (Kitty Cocktail Bar)

판타지 세계의 **고양이 바텐더이자 연금술사**가 되어, 손님의 주문에 맞춰
재료를 고르고 **휴대폰을 직접 흔들어** 마법 물약과 칵테일을 제조하는 모바일 웹 게임입니다.

> 핵심 재미는 "얼마나 많이 흔드느냐"가 아니라 **"정확한 방식으로 흔드는 것"** 입니다.

## 🎮 플레이 방법

1. **손님 주문 확인** — 손님이 원하는 효과를 듣고 어떤 물약이 필요한지 추론합니다.
   (예: "용과 싸워야 하는데 불이 무서워요" → 화염 저항 물약)
2. **재료 선택** — 선반에서 필요한 재료를 골라 담습니다. 잘못 넣어도 게임은
   끝나지 않고, 이상한 결과물이 나옵니다.
3. **제조 조작** — 재료마다 조작이 다릅니다.
   - 🍄 과일·허브: 화면을 **탭해서 으깨기**
   - 💧 액체: **길게 눌러 따르기**
   - ✨ 가루: **톡톡 두드려 넣기**
4. **흔들기** — 셰이커를 흔들어 섞습니다. 세 가지를 동시에 맞춰야 합니다.
   - **강도**: 목표 존 안에 바늘을 유지 (약하게 / 적당히 / 강하게)
   - **리듬**: 박자 표시가 중앙선에 닿을 때 흔들기
   - **멈추기**: 거품이 병목선에 닿기 직전에 `STOP` — 늦으면 뚜껑이 날아갑니다!
5. **결과 판정** — 재료 정확도·강도·리듬·멈춤 타이밍을 종합해 등급이 결정됩니다.

## 🏆 결과 등급

- 🌟 완벽한 제조
- ✨ 훌륭한 물약
- 😌 맛은 좋지만 효과가 약함
- 💥 효과는 강하지만 부작용 발생
- ❓ 정체불명의 물약
- 🧨 펑! 폭발!

실패는 벌점이 아니라 **웃긴 콘텐츠**가 됩니다. (목소리가 고양이 울음으로 변하거나,
투명해졌는데 옷만 남거나, 손님이 잠시 닭으로 변하는 등)

## 🧩 게임 구조

- **스테이지형**: 하루에 여러 손님을 상대하고, 만족도에 따라 골드·평판을 얻습니다.
- **성장 요소 (상점)**: 골드로 장비를 업그레이드합니다.
  큰 셰이커 · 자동 계량 도구 · 실수 방지 부적 · 고급 잔.
- **손님 수집**: 기사(강한 효과)·마법사(정확한 제조)·고블린(이상한 부작용)·
  뱀파이어(붉은 재료)·유령(부드러운 제조)마다 선호가 다릅니다.

## 📱 조작 & 실행

별도의 빌드나 설치가 필요 없는 정적 웹 게임입니다.

```
# 로컬에서 실행 (아무 정적 서버나 사용 가능)
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

- **모바일**: 실제로 휴대폰을 흔들어 조작합니다. (iOS는 첫 실행 시 모션 센서 권한 허용 필요)
- **데스크톱**: 모션 센서가 없으면 자동으로 대체 입력이 활성화됩니다.
  화면을 빠르게 **드래그**하거나 **Space** 키를 연타해 흔들기를 대신합니다.

## 🗂️ 구조

```
index.html      진입점 (타이틀 화면 포함)
css/style.css   전체 스타일 (귀엽고 유쾌한 어두운 마법 술집 테마)
js/data.js      재료 · 손님 · 주문(레시피) · 부작용 · 업그레이드 데이터
js/shake.js     흔들기 감지 (DeviceMotion + 데스크톱 대체 입력)
js/ui.js        화면 전환 및 제조 미니게임
js/game.js      게임 상태 머신 및 결과 판정 로직
```

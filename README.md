# 🐱🍸 고양이 판타지 쉐이크 (Kitty Cocktail Bar)

판타지 세계의 **고양이 바텐더**가 되어, 손님의 주문에 맞는 재료를 넣고
**휴대폰을 직접 흔들어** 마법 칵테일을 제조하는 모바일 웹 게임입니다.

> 손님이 기다려주는 **제한 시간** 안에, 칵테일마다 정해진 **쉐이킹 횟수**를 채워야 합니다!

## 🎮 플레이 방법

1. **손님 주문 확인** — 손님이 원하는 칵테일과 그 칵테일의 **필요 재료 수 · 쉐이킹 횟수 ·
   제한 시간**을 확인합니다. `주문 받기`를 누르면 대기 시간이 흐르기 시작합니다.
2. **재료 선택** — 레시피에 맞는 재료를 골라 담습니다. 잘못 넣으면 정체불명의 칵테일이
   나옵니다. (게임은 끝나지 않아요!)
3. **흔들기** — 휴대폰을 흔들어 칵테일을 완성합니다.
   - 칵테일마다 필요한 쉐이킹 횟수가 다릅니다. (예: 딸기 셰이크 6회, **번개 칵테일 100회**)
   - 손님의 **대기 시간이 다 되기 전에** 필요한 횟수를 모두 채우면 완성!
   - 시간 안에 못 채우면 **주문 실패** — 손님이 화나서 떠납니다.
4. **결과 판정** — 완성 여부, 재료 정확도, 남은 시간을 종합해 등급이 결정됩니다.

## 🏆 결과 등급

- 🌟 완벽한 칵테일 — 정확한 재료 + 여유 있게 완성
- ✨ 훌륭한 칵테일
- 😌 맛은 좋지만 평범함
- ❓ 정체불명의 칵테일 — 재료가 엉망일 때 (웃긴 부작용 발생!)
- 💢 주문 실패 — 제한 시간 초과

정체불명의 칵테일은 벌점이 아니라 **웃긴 콘텐츠**가 됩니다. (목소리가 고양이 울음으로
변하거나, 투명해졌는데 옷만 남거나, 손님이 잠시 닭으로 변하는 등)

## 🧩 게임 구조

- **스테이지형**: 하루에 여러 손님을 상대하고, 만족도에 따라 골드·평판을 얻습니다.
- **성장 요소 (상점)**: 골드로 장비를 업그레이드합니다.
  큰 셰이커(흔들기 효율↑) · 자동 계량 도구 · 모래시계 부적(제한 시간↑) ·
  실수 방지 부적 · 고급 잔.
- **손님 수집**: 기사(힘든 칵테일 선호)·마법사/용사(정확한 배합)·고블린(이상한 결과)·
  뱀파이어(붉은 재료)·유령(여유로운 완성)마다 선호가 다릅니다.
  **탐정 고양이**는 8프레임 스프라이트 애니메이션으로 등장합니다. 🔍

## 📱 조작 & 실행

별도의 빌드나 설치가 필요 없는 정적 웹 게임입니다.

```
# 로컬에서 실행 (아무 정적 서버나 사용 가능)
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

- **모바일**: 실제로 휴대폰을 흔들어 조작합니다. (iOS는 첫 실행 시 모션 센서 권한 허용 필요)
- **데스크톱**: 모션 센서가 없으면 자동으로 대체 입력이 활성화됩니다.
  화면을 빠르게 **드래그**하거나 **Space** 키를 연타해 흔들기를 대신합니다.

## 🗂️ 구조

```
index.html            진입점 (타이틀 화면 포함)
assets/               스프라이트 등 이미지 에셋 (탐정 고양이 8프레임 스트립)
css/style.css   전체 스타일 (귀엽고 유쾌한 어두운 마법 술집 테마)
js/data.js      재료 · 손님 · 주문(레시피) · 부작용 · 업그레이드 데이터
js/shake.js     흔들기 감지 (DeviceMotion + 데스크톱 대체 입력)
js/ui.js        화면 전환 (주문 · 재료 선택 · 흔들기 · 결과)
js/game.js      게임 상태 머신 · 대기 시간 타이머 · 결과 판정 로직
```
